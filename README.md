# 911 Local Plumbing — Permit Tracker

React + Vite + TypeScript + Supabase.

## Setup

1. `npm install`
2. Copy `.env.local.example` to `.env.local` and fill in `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
3. Run the migration in `supabase/migrations/20260427_init.sql` against your Supabase project.
4. `npm run dev` — http://localhost:5173
